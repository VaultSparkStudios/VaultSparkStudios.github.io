Open Source & Licenses

VaultSpark Studios builds in the open where we can. Below is a list of our public repositories, the licenses they ship under, and attributions for third-party software we use across the studio.

The VaultSpark Studios website — the site you are on right now. Static HTML/CSS/JS, hosted on GitHub Pages.

Third-Party Attributions

VaultSpark Studios projects use the following open-source software and services. We are grateful to these communities.

Website & Frontend

  • Google Analytics (GA4) — analytics and usage measurement
  • Supabase — authentication, database, and real-time services (MIT / Apache 2.0)
  • Sentry — error tracking and performance monitoring
  • Web3Forms — contact form submission API

Build & Development

  • Node.js — JavaScript runtime (MIT)
  • Playwright — end-to-end testing framework (Apache 2.0)
  • Vite — frontend build tool (MIT)
  • React — UI library (MIT)
  • TypeScript — typed JavaScript (Apache 2.0)

Infrastructure

  • GitHub Pages — static site hosting
  • GitHub Actions — CI/CD workflows
  • Cloudflare — DNS and CDN services
