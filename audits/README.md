# Public Audit Boundary

This repository no longer ships detailed internal audit artifacts.

Why:
- audit files can expose internal scoring, backlog prioritization, operating notes, and launch-readiness detail that belongs in private studio operations

Public-safe rule:
- keep only high-level public status in this repo
- keep detailed audits, internal scoring, and operating analysis in the private Studio OS / ops repository

Local private backup:
- a local backup of the removed audit artifacts was preserved outside this repo during the sanitization pass on 2026-04-03
