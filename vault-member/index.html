ENTITY REPORT // DS-0 // EYES ONLY

Subject designation DreadSpike is not a single entity. Field analysts tracking the Sovereign Wraith phenomenon have confirmed three simultaneous manifestations across unconnected sites within a 72-hour window.

Each instance exhibited full behavioral coherence. No evidence of communication between sites. Current hypothesis: the signal is the entity. Containment status — unresolved.

ORIGIN ANOMALY // UNRELEASED

The Vault's original designation for DreadSpike was The First Fault — a fracture in the boundary layer between the Vault's constructed reality and whatever precedes it. The name was redacted from all public records before the universe launched.

Recovery teams found the entity already self-named. Every recovered artifact had the same glyph carved into it. The translation came back: DreadSpike.

BEHAVIORAL NOTE // ACTIVE THREAT

DreadSpike does not pursue. It positions. Field teams repeatedly encountered the entity already in place — as if it had knowledge of where the teams would be before they did.

Two analysts refused to file their reports. One stated: "It was waiting for us because it had already read what we were going to write." Status of both analysts — reassigned.

NEXT SIGNAL // WHAT'S COMING

The DreadSpike chapter in the Vault Universe is not a standalone arc. The entity is woven through the foundation of every world we're building — present in Dunescape's mythic layer, referenced in VaultFront's intel reports, and structurally embedded in what MindFrame is about.

Vault Members will receive the next classified file before it becomes public. The next drop: The Cartographers — those who mapped DreadSpike and didn't return whole.
