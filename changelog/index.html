Vault Changelog

Every feature, fix, and improvement shipped to the vault and public site — written for members, not engineers.

Phase 44–49March 2026
Social Graph, Treasury, Seasons & Weekly Leaderboard
  • Social graph: follow and unfollow any vault member — following count displayed on profiles
  • Activity feed: live signal updates filtered to members you follow
  • Vault Treasury: points economy with a rewards catalog members can redeem against
  • Seasons XP system: time-limited progression seasons with XP tracking and rank tiers
  • Weekly leaderboard tab: resets every Monday, highlights top XP earners for the current week
  • Demo infrastructure: game embed slots and placeholder sections added to all game pages
  • Homepage: Vault Live section for streaming dev sessions direct from the front page
  • Journal: "Coming From The Forge" sidebar showing planned upcoming dispatches
Phase 43March 2026
Co-op Teams
  • Co-op teams: create a team or join one with a 6-character invite code
  • Team points accumulate automatically as members earn XP
  • Team leaderboard tab on /leaderboards/ — ranked by total points with medal icons
  • Team panel in the vault: create, join, leave, or disband your team
Phase 40–42March 2026
Fan Art System & Community Badges
  • Fan art submission: Vault Members can upload artwork (PNG/JPG/WebP) with title, description, and character tag
  • Fan art gallery on /community/ — browse approved submissions in a card grid
  • Fan art voting: heart/like buttons with live vote counts; Vault Members can toggle their vote
  • Fan art moderation queue in Vault Command for approved/rejected submissions
  • "Currently Playing" badge on public member profiles — shows active or last-played game
Phase 36–39March 2026
Game Score API, Sessions & Performance
  • Game session recording: visiting a game page as a signed-in member logs a session (once per day)
  • Session milestone awards at 5, 10, and 25 sessions per game (25/50/100 pts)
  • Game Score API live: vault-score.js SDK lets game pages submit scores directly to the vault leaderboard
  • WebP image conversion: DreadSpike hero images converted to WebP for faster page loads
  • Supabase query batching: portal startup now makes one round-trip instead of several (faster load)
  • Per-page OG images: 14 unique branded share images across the site (no more generic previews)
Phase 33–35March 2026
CI Quality Gates, Events & Investor Tools
  • Lighthouse CI score gate: every push is tested for performance and accessibility automatically
  • Axe-core accessibility CI: 7 public pages scanned for WCAG violations on every deploy
  • Community event RSVPs: RSVP buttons on all event cards with live count
  • Investor data room access log: every document open is recorded for the founder
  • Rate limiting on invite code claims: prevents registration spam (client-side 3 attempts / 10 min)
  • Vault Command: member CSV export and challenge analytics dashboard
  • Gift Points: member-to-member point gifting panel in the vault
Phase 29–32March 2026
Dark Mode, Ratings & Profile Themes
  • Dark/light mode toggle: ☀/🌙 button on every public page, persists across sessions
  • Profile card themes: 5 rank-unlockable card backgrounds (Rift Blue, Void, Forge Fire, Sparked)
  • Game ratings: 5-star widget on all released game pages; star average in JSON-LD schema for search results
  • Journal view counts: each post now shows how many times it has been read
  • Auto-generated sitemap: sitemap.xml rebuilds on every push to main
Phase 25–28March 2026
Game Hub, Offline Mode & Member Tools
  • Games hub: genre filter pills (Sports, Action, Strategy, MMORPG, Puzzle) alongside the existing status filter
  • Offline detection banner in the vault portal — shows when your connection drops
  • Member Spotlight panel: random top-20 member shown on each login
  • Rank Comparison panel: see your points gap to the next player above you on the leaderboard
  • Referral QR code: one click generates a QR code for your referral link
  • Login activity heatmap: 12-week GitHub-style grid in the Chronicle tab
  • Annual vault anniversary: awards 50 pts × years on your join date
  • Weekly XP recap banner: shows last week's total on Mondays (dismissible)
  • Vault Command: scheduled signal broadcasts — set a future date/time for studio updates
Phase 22–24March 2026
VaultSparked Page, Analytics & Onboarding
  • VaultSparked perks landing page: /vaultsparked/ with full pricing, perk grid, and comparison table
  • Custom 404 page: rank-themed with quick links and a CTA to join the vault
  • Investor portal: KPI sparklines with 30-day trend lines under each metric
  • Investor portal: "Ask a question" card — questions go directly to the founder
  • Lazy-load audit: loading="lazy" added to all non-critical images across 29 public pages
Phase 20–21March 2026
Vault Command + Performance
  • Vault Command: challenge analytics dashboard (completions per challenge, ranked by popularity)
  • Vault Command: member CSV export (one-click download of all members)
  • Challenges tab: collapsible completion history showing when each challenge was earned
  • Settings: password reset email trigger (no more digging for the forgot-password link)
  • Settings: GDPR data export (downloads your full data as JSON)
  • Settings: account deletion request flow
  • Supabase preconnect hints added to all pages that use the API
  • Public changelog page (you're reading it!)
  • security.txt added at /.well-known/security.txt
Phase 17–19March 2026
Game Pages + Lore Archive + Privacy
  • Game pages: Vault Points callout added to Call of Doodie, Gridiron GM, and VaultSpark Football GM
  • Unreleased game pages: "Notify Me at Launch" wishlist section (Solara, MindFrame, Project Unknown, VaultFront)
  • Classified Archive: reading progress bar (X of N files read)
  • Classified Archive: live keyword search across all files
  • Classified Archive: bookmark files (🔖) with localStorage persistence and filter toggle
Phase 15–16March 2026
Challenge Streaks, Community + Member Directory
  • Challenge streaks: 🔥 badge shows your consecutive-day streak on the Challenges tab
  • Streak milestone bonuses: 7-day (+50 pts) and 30-day (+200 pts)
  • Challenge difficulty tiers: Easy / Medium / Hard / Legendary badges on every challenge card
  • First-completion micro-achievements: 1st, 5th, and 10th challenge each award bonus pts
  • Leaderboards: ⚡ Challenges tab (top members by completions) and 🔗 Recruiters tab (top referrers)
  • Community page: recently joined member chips and live community polls section
  • Member directory: new /members/ page with rank filter and search
  • Polls: 🗳️ Polls tab in the vault for voting on studio decisions
Phase 12–14March 2026
Journal, PWA, Dashboard Upgrades
  • Individual journal post pages with full SEO, Article JSON-LD, author bio, reading time
  • Journal tag filtering (Studio OS / Games / Launch / Origin / Studio / Site)
  • Journal RSS feed updated to per-post canonical URLs
  • PWA install prompt — install the vault as a home screen app
  • Push notification opt-in with improved 📡 value-prop banner in Settings
  • Dashboard tab persistence — portal remembers your last active tab across sessions
  • "What's New" modal — shows unread Studio Pulse entries on login
  • Points breakdown modal — click "breakdown" on your Vault Points stat
  • Challenge completion modal — replaced XP chip with full slide-up panel
  • Onboarding tour now saved to your Vault profile
  • Points history chart — 30-day bar chart on the Chronicle tab (pure SVG, no dependencies)

  • 5-step onboarding tour for new vault members
  • In-portal notification center (🔔 bell, Studio Pulse realtime feed)
  • Challenge categories + filter pills
