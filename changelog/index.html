Vault Changelog

Every feature, fix, and improvement shipped to the vault and public site — written for members, not engineers.

Phase 20–21March 2026
Vault Command + Performance
  • Vault Command: challenge analytics dashboard (completions per challenge, ranked by popularity)
  • Vault Command: member CSV export (one-click download of all members)
  • Challenges tab: collapsible completion history showing when each challenge was earned
  • Settings: password reset email trigger (no more digging for the forgot-password link)
  • Settings: GDPR data export (downloads your full data as JSON)
  • Settings: account deletion request flow
  • Supabase preconnect hints added to all pages that use the API
  • Public changelog page (you're reading it!)
  • security.txt added at /.well-known/security.txt
Phase 17–19March 2026
Game Pages + Lore Archive + Privacy
  • Game pages: Vault Points callout added to Call of Doodie, Gridiron GM, and VaultSpark Football GM
  • Unreleased game pages: "Notify Me at Launch" wishlist section (Dunescape, MindFrame, Project Unknown, VaultFront)
  • Classified Archive: reading progress bar (X of N files read)
  • Classified Archive: live keyword search across all files
  • Classified Archive: bookmark files (🔖) with localStorage persistence and filter toggle
Phase 15–16March 2026
Challenge Streaks, Community + Member Directory
  • Challenge streaks: 🔥 badge shows your consecutive-day streak on the Challenges tab
  • Streak milestone bonuses: 7-day (+50 pts) and 30-day (+200 pts)
  • Challenge difficulty tiers: Easy / Medium / Hard / Legendary badges on every challenge card
  • First-completion micro-achievements: 1st, 5th, and 10th challenge each award bonus pts
  • Leaderboards: ⚡ Challenges tab (top members by completions) and 🔗 Recruiters tab (top referrers)
  • Community page: recently joined member chips and live community polls section
  • Member directory: new /members/ page with rank filter and search
  • Polls: 🗳️ Polls tab in the vault for voting on studio decisions
Phase 12–14March 2026
Journal, PWA, Dashboard Upgrades
  • Individual journal post pages with full SEO, Article JSON-LD, author bio, reading time
  • Journal tag filtering (Studio OS / Games / Launch / Origin / Studio / Site)
  • Journal RSS feed updated to per-post canonical URLs
  • PWA install prompt — install the vault as a home screen app
  • Push notification opt-in with improved 📡 value-prop banner in Settings
  • Dashboard tab persistence — portal remembers your last active tab across sessions
  • "What's New" modal — shows unread Studio Pulse entries on login
  • Points breakdown modal — click "breakdown" on your Vault Points stat
  • Challenge completion modal — replaced XP chip with full slide-up panel
  • Onboarding tour now saved to your Vault profile
  • Points history chart — 30-day bar chart on the Chronicle tab (pure SVG, no dependencies)

Public Pages, SEO, Community Features
  • New public pages: /leaderboards/, /community/, /journal/archive/, /ranks/, /member/, /status/, /search/
  • Daily login bonus + streak system (🔥 day counter, milestone bonuses at 7/30/60/100 days)
  • 5-step onboarding tour for new vault members
  • In-portal notification center (🔔 bell, Studio Pulse realtime feed)
  • Challenge categories + filter pills
  • Seasonal challenges with countdown badges
  • Achievement progress bars
