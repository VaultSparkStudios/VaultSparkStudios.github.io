VaultFront

View Project Get Early Access

A Game and a Platform.

VaultFront is both a real-time strategy game and a live web platform. The game: vault capture mechanics, convoy timing, asymmetric pressure. The platform: a dedicated backend runtime being built on VaultSpark's studio VPS infrastructure — Docker Compose, Caddy, WebSocket play layer, and a dedicated API origin.

  • Real-time vault capture game mechanics — convoy timing and chokepoint control
  • Dedicated backend runtime (play-vaultfront.vaultsparkstudios.com) in development
  • API origin (api-vaultfront.vaultsparkstudios.com) being brought online
  • Browser-native — built for the web from the ground up, no client download
  • Part of VaultSpark's shared studio infrastructure model (VPS + Docker + Caddy)
  • Early access through Vault Membership
