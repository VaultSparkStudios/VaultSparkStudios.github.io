# Project Brief

## Identity

- Name: VaultSpark Studios Website
- Slug: vaultsparkstudios-website
- Type: Studio website + Vault Member portal (web app)
- Status: Live
- Owner: VaultSpark Studios (p4cka)

## Why this exists

- Problem or opportunity: VaultSpark Studios needs a public-facing home and a members-only portal that rewards community engagement, surfaces classified lore, and gates early access to games.
- Audience: Game enthusiasts, early adopters, lore hunters, supporters of indie studios.
- Studio importance: Primary public brand surface and the only system that tracks member identity, rank, points, and subscriptions across all studio projects.

## Success

- What success looks like: Members log in regularly, rank up through activity, unlock lore, claim beta keys, and feel part of an exclusive inner circle. Subscribers automatically receive the VaultSparked Discord role. Admins can broadcast Studio Pulse messages and drop beta keys from the portal.
- What this project is not: Not a game itself. Not a generic blog or CMS. Not a social network.

## Scope

- In scope: Auth (email, Google, Discord OAuth), Vault Member portal (rank, points, challenges, classified files, beta keys, referrals, Studio Pulse, Discord role sync), admin panel (Vault Command), Stripe subscription integration, Supabase backend, Edge Functions.
- Out of scope: Game engines or game builds, Discord bot server management, email marketing platform.
